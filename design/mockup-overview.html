SchoolSync

A mobile app that helps parents stay on top of their children's school life. It reads school emails, extracts events and tasks using AI, and presents everything in an organized dashboard with calendars, to-do lists, and intelligent notifications.

Key Features

SchoolSync has six main areas that parents interact with daily:

📊 Home Dashboardscreen

Priority-first view: overdue items, this week's events, key highlights, and briefing cards when new school information arrives.

📅 Calendarscreen

Monthly view with colored dots per child. Scrollable event list with filters by child or date range.

✅ To-Do Listsscreen

Tasks grouped by urgency — overdue, urgent, this week, later. Filters per child with progress tracking.

💬 AI ChatAI

Ask questions in natural language: "What's due this week for Emily?" and get answers from the AI assistant.

📄 Document UploadAI

Photograph school letters, newsletters, or screenshots. AI extracts events, tasks, and key information automatically.

👦 Children Profilesscreen

Each child's school details, class lists, key contacts, and reference information in one place.

Behind the Scenes

The app has 11 cloud functions running on Google's servers. These handle everything that can't happen on the phone itself:

🤖 AI Processing

Forwards requests to Google Gemini AI. Rate-limited to 20 requests/minute per user. Never exposes the API key to the phone.

📬 Daily Summaries

Sends a morning push notification summarizing what's due today. Parents choose their delivery time. Auto-reschedules daily.

🔔 Smart Reminders

Evening notifications about tomorrow's events and tasks. Personalized per parent's actual data. Duplicate protection built in.

🔒 Encryption

All user data encrypted in the database with a unique key per account. Decrypted only when the user signs in.
