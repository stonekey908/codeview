Description
Shows top bilateral trade routes worldwide, with search, sorting by volume/growth/tariff, and a detail popup for each corridor's commodities and statistics.
✨ AI Explanation
The main trade corridors view showing the top 20 bilateral trade routes worldwide. Users can search corridors, sort by volume/growth/tariff rate, and click any corridor to see a detail popup with commodity breakdowns and statistics.

This is the default landing page — when users navigate to the dashboard root, they get redirected here automatically.

It fetches data from the Trade Flows API, which either calls Google Gemini for live data or falls back to a researched static dataset.
