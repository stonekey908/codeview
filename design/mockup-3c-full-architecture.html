Shows top bilateral trade routes worldwide, with search, sorting by volume/growth/tariff, and a detail popup for each corridor.
✨ AI Explanation
The main trade corridors view showing the top 20 bilateral trade routes worldwide. Users can search, sort by volume/growth/tariff, and click any corridor to see commodity breakdowns and statistics.

This is the default landing page — the dashboard redirects here automatically. It fetches data from the Trade Flows API, which calls Google Gemini or falls back to static data.
